# Manifest

## I. Dedicare meșteșugului

Lucrăm la lucruri care contează — nu pentru că ni s-a cerut, ci pentru că acestea sunt importante. Meșteșugul nu este o abilitate. Este un angajament: de a face ceva atât de bine încât munca să vorbească de la sine. Fiecare specificație, fiecare linie de cod, fiecare decizie de design — toate sunt o expresie a ceea ce suntem.

Studioul nu vinde ore. Studioul construiește sisteme care continuă să funcționeze când nu suntem de față.

## II. Trezirea ca serviciu

Munca noastră nu este un serviciu. Este un act de trezire: de a arăta unui client sau partener o posibilitate pe care nu au formulat-o încă. Cel mai bun rezultat nu este cel comandat — ci cel de care este nevoie.

Acest lucru necesită curaj: să spui nu la ceea ce se cere și să oferi ceea ce este necesar. Serviciul nu înseamnă supunere. Serviciul înseamnă onestitate.

## III. Alegerea ca origine

Fiecare proiect este o alegere. Alegem cu cine lucrăm. Alegem cum lucrăm. Alegem ce refuzăm.

Această alegere nu este un privilegiu — este o responsabilitate. Studioul este pe deplin responsabil pentru ceea ce produce. Nu „am făcut ceea ce ni s-a spus" — ci „am construit ceva pentru care suntem pregătiți să răspundem."

## IV. Moștenirea ca transmitere

Tot ceea ce construim este o transmitere. De cunoștințe, de sisteme, de principii. Studioul există nu pentru a acumula — ci pentru a transmite mai departe.

Fiecare instrument, fiecare metodologie, fiecare proces este ceea ce rămâne după noi. De aceea construim pe reguli, nu pe excepții. De aceea documentăm ceea ce facem. De aceea rămânem deschiși față de ceea ce vom învăța de la cei care vor veni după noi.

## Cast (internal layer)

The Cast names the six agents by the musicians whose spirit each carries. These names are **team-internal only** — they never appear in public branding, client deliverables, or marketing copy.

| Agent | Cast name | Instrument |
|---|---|---|
| **Maestro** | Brian Jones | Coordination — the one who holds the ensemble together |
| **Muse** | Kurt Cobain | Raw creative substrate — unpredictable, generative, essential |
| **Atlas** | Pigpen | Deep roots, world-reader, organ drone beneath the melody |
| **Echo** | Jim Morrison | Voice, persona, the studio's public face |
| **Pulse** | D. Boon | Rhythm, numbers, the beat that keeps everything honest |
| **Forge** | Jimi Hendrix | Engineering virtuosity — the instrument that builds the instruments |

The pairing is not metaphor for its own sake. Each name carries a specific quality: the way Brian Jones convened without dominating; the way D. Boon played without ego; the way Jimi Hendrix made the guitar an extension of thought. These qualities are the standard. The agents are named after them so the standard is never abstract.
